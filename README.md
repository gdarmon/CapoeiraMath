# Darmon Ginga

משחק PWA קטן לאנדרויד לתרגול לוח הכפל 7, 8 ו-9 דרך קפוארה, קצב ותשובות בלחיצה.

## הפעלה מקומית

```bash
cd /localdrive/users/giladdar/app_bazel/darmon-ginga
npm start
```

פתחו בדפדפן:

```text
http://localhost:4173
```

## התקנה באנדרויד

כדי להתקין על הטלפון, צריך לפתוח את האפליקציה ב-Chrome מכתובת HTTPS, או להגיש אותה משרת מקומי עם תעודה. אחרי שהדף נטען:

1. פותחים את תפריט Chrome.
2. בוחרים `Install app` או `Add to Home screen`.
3. האפליקציה תמשיך לעבוד גם ללא אינטרנט אחרי הטעינה הראשונה.

## מה יש במשחק

- רודות קצרות של 12 שאלות.
- מצב 7, מצב 8, מצב 9 ומצב בוס 7-8-9.
- אין הקלדה: רק כפתורי תשובה גדולים.
- חזרה חכמה על טעויות.
- אנימציית קפוארה, צלילי קצב, רטט והקראה בעברית.
- שמירת התקדמות מקומית במכשיר.
